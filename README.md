# NBA Player and Team Performance Analysis

## Project Overview

This project uses SQL to analyze NBA player, team, game, draft, and performance data.

The goal is to build a structured relational database and answer analytical questions related to team performance, player roles, draft outcomes, and game results.

## Tools Used

- SQL
- Relational database design
- Data cleaning
- Joins
- Aggregations
- Common table expressions
- Window functions
- Ranking and filtering

## Datasets

The project uses several NBA datasets, including:

- Game data
- Team information
- Player information
- Line scores
- Draft history
- Draft combine statistics
- Common player information

## Questions Explored

Examples of questions this project will answer include:

1. Which teams had the strongest performance by season?
2. Which players ranked highest in key performance categories?
3. How did home teams perform compared with away teams?
4. How does draft position relate to later player performance?
5. Which teams performed best in close games?
6. Which players had the largest roles on successful teams?
